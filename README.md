# 🚀 Astro Project

A modern web project built using the Astro framework for fast and optimized performance.

## 🚀 Live Demo

👉 https://astro-project-wheat.vercel.app/

## 📌 About the Project

This project is built using Astro, a modern static site generator that focuses on performance and minimal JavaScript. It helps in creating fast, SEO-friendly, and highly optimized websites.

## 💡 Features

* ⚡ Fast loading performance
* 📦 Optimized build with minimal JavaScript
* 🎯 SEO-friendly structure
* 📱 Responsive design
* 🧩 Component-based architecture

## 🛠️ Tech Stack

* Astro
* HTML
* CSS
* JavaScript
